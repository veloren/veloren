# Contributing

Issues and pull requests are welcome on the [Nova-Forge GitHub repository](https://github.com/shifty81/Nova-Forge).

## Focus Areas

- LAN discovery and connection UX
- Auth-free server stability and compatibility
- Singleplayer world management
- Performance on lower-end hardware

## Guidelines

### Do not commit tainted content to the repository

If you commit anything that was not created by yourself, double check that the license for that content permits distribution under the GPL-3.0 license.
